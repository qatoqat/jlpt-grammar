JLPT Grammar Tree

Particles — Multiple Uses

Topic Marker は (wa)

Marks the topic; “as for…”

わたしは学生です。I am a student.
See also: Contrast
Contrast Marker は (wa)

Implies contrast with other options (often in negatives).

肉は食べますが、魚は食べません。I eat meat, but I don’t eat fish.
See also: Topic
Subject Marker が (ga)

Marks the subject (focus/emphasis).

これが本です。This is the book (the one that is).
See also: But (conjunction)
But / However が (ga)

Connects clauses with contrast (formal).

すみませんが、行けません。Sorry, but I can’t go.
See also: Subject marker
Object Marker を (o)

Marks the object of an action.

ご飯を食べる。Eat a meal.
Direction / Destination に (ni)

To; towards (destination).

日本に行く。Go to Japan.
See also: Time (specific) See also: Existence place
Time (specific) に (ni)

At a specific time.

7時に起きる。Wake up at 7.
See also: Direction
Existence (animate) に (ni) + いる

Indicates location of animate existence.

庭に猫がいる。There is a cat in the garden.
See also: Inanimate existence (が・ある)
Indirect Object (Receiver) に (ni)

Marks the recipient in giving/teaching verbs.

友達に本をあげる。Give a book to a friend.
See also: Direction
Place of Action で (de)

Where an action takes place (at/in).

図書館で勉強する。Study at the library.
See also: Means/Method See also: Reason
Means / Method で (de)

By means of; with; using.

日本語で書く。Write in Japanese.
See also: Place of action
Total Amount / Time Limit で (de)

In (total amount/time); within.

全部で1000円です。It’s 1000 yen in total.
See also: Reason
Reason / Cause で (de)

Because of (cause/reason).

病気で休む。Rest because of illness.
See also: ので (polite reason)
Possessive / Modifier の (no)

's; connects nouns.

私の本。My book.
See also: Nominalizer
Verb Nominalizer の (no)

Turns a clause/verb into a noun.

走るのが好き。I like running.
See also: Possessive
Question Particle か (ka)

Makes a sentence a question.

行きますか。Will you go?
See also: Or (embedded questions)
Or (Choice) か (ka)

Or; indicates a choice or uncertainty.

コーヒーか紅茶。Coffee or tea.
See also: Question
From (Origin) から (kara)

Starting point/origin.

日本から来た。Came from Japan.
See also: Because
Because (Reason) から (kara)

Because, since (casual).

寒いから行かない。I won’t go because it’s cold.
See also: From See also: ので
Until / To (Extent) まで (made)

Up to (time/place); as far as.

5時まで待つ。Wait until 5.
See also: By (deadline) – までに
Direction (towards) へ (e)

Towards (directional emphasis).

学校へ行く。Go towards school.
See also: Destination (に)
And / With と (to)

And (listing nouns); with.

友達と行く。Go with a friend.
Also / Too も (mo)

Also; too; as well.

私も行く。I’m going too.
And (partial list) や (ya)

And; non-exhaustive list.

リンゴやミカン。Apples, oranges, etc.

Copula & Basic Sentences

To be (polite) です (desu)

Is/are/am (polite copula).

学生です。I am a student.
To be (casual) だ (da)

Is/are/am (plain copula).

雨だ。It’s raining.
Is not (polite) じゃない / ではない (ja nai / de wa nai)

Is not (negative of です/だ).

学生ではない。Not a student.

Verbs (Basic Forms)

Polite Present/Future ～ます (masu)

Will do / do (polite non-past).

行きます。Will go.
Polite Past ～ました (mashita)

Did (polite past).

行きました。Went.
Polite Negative ～ません (masen)

Will not do / do not (polite).

行きません。Won’t go.
Te-form (connect/do) ～て (te)

Connects actions; basis for requests.

本を読んで、寝る。Read a book and sleep.
Request (Please do) ～てください (te kudasai)

Please do …

座ってください。Please sit down.
Progressive / State ～ている (te iru)

Doing now; resulting state.

今食べている。Eating now.
After doing ～てから (te kara)

After doing X.

ご飯を食べてから行く。Go after eating.
Please don’t ～ないでください (naide kudasai)

Please don’t do …

入らないでください。Please don’t enter.
Must not ～てはいけない (te wa ikenai)

Must not do …

ここに入ってはいけない。Must not enter here.
Want to do ～たい (tai)

Want to do …

行きたい。Want to go.
Have done before ～たことがある (ta koto ga aru)

Have done (experience).

日本に行ったことがある。Have been to Japan.
Plan / Intend to ～つもりだ (tsumori da)

Plan to; intend to.

行くつもりだ。Plan to go.

Adjectives

i-adj present ～い (i)

Is [adj].

大きい。Big.
i-adj negative ～くない (kunai)

Is not [adj].

大きくない。Not big.
i-adj past ～かった (katta)

Was [adj].

大きかった。Was big.
na-adj present ～な / だ (na / da)

Is [adj] (na-adj).

静かだ。Quiet.
na-adj negative ～じゃない (ja nai)

Is not [na-adj].

静かじゃない。Not quiet.

Connectors & Others

Because (softer) ～ので (node)

Because, since (polite).

寒いので行かない。Since it’s cold, I won’t go.
See also: から (because)
But / However ～けど / けれど (kedo / keredo)

But, however (casual).

安いけど、古い。It’s cheap but old.
See also: が (but)
Comparison ～より～のほうが (yori ~ no hou ga)

More X than Y.

歩くよりバスのほうが速い。The bus is faster than walking.
Existence (inanimate) ～がある (ga aru)

There is (a thing).

庭に木がある。There is a tree in the yard.
See also: Animate existence (に・いる)

Conditionals & Time

If (conditional) ～ば (ba)

If … then ….

行けば、会える。If you go, you can meet.
When / If (after) ～たら (tara)

When/if it happens ….

家に着いたら連絡する。When I get home, I’ll contact you.
In the case that ～場合は (baai wa)

In the event/ case that ….

雨の場合は中止だ。In case of rain, it’s cancelled.
If, in case ～なら (nara)

If (topic/conditional).

京都なら、新幹線が便利だ。If it’s Kyoto, the shinkansen is convenient.
After, later ～あとで (ato de)

After; later.

ご飯のあとで勉強する。Study after the meal.
By (deadline) ～までに (made ni)

By (a certain time).

5時までに来て。Come by 5 o’clock.
See also: Until (N5)
While / During ～間 (aida)

While; during.

映画の間、寝ていた。Slept during the movie.
While doing ～ながら (nagara)

While (doing two things).

音楽を聴きながら勉強する。Study while listening to music.

Possibility, Ability & Necessity

Can do ～ことができる (koto ga dekiru)

Can; be able to.

日本語を話すことができる。Can speak Japanese.
Must do ～なければならない (nakereba naranai)

Must; have to.

勉強しなければならない。Must study.
Don’t have to ～なくてもいい (nakute mo ii)

Don’t have to; it’s okay not to.

行かなくてもいい。You don’t have to go.
Want (something) ～がほしい (ga hoshii)

Want something.

新しい車がほしい。Want a new car.
Purpose (go to do) ～に (ni)

Go/come for the purpose of.

本を買いに行く。Go to buy a book.
See also: Direction (N5)

Likelihood, Guess & Concession

Might / Maybe ～かもしれない (kamoshirenai)

Might be; maybe.

雨かもしれない。It might rain.
Should be / Must be ～はずだ (hazu da)

Should be; expected to be.

彼は来るはずだ。He should come.
Cannot be ～はずがない (hazu ga nai)

Cannot be; unlikely.

そんなはずがない。That can’t be.
Although / Even though ～のに (noni)

Despite; even though.

勉強したのに、失敗した。Even though I studied, I failed.
Just by ～だけで (dake de)

Just by doing ….

これだけでいい。This alone is enough.
Only, nothing but ～ばかり (bakari)

Only doing X; full of.

ゲームばかりしている。Doing nothing but playing games.
Like / Similar ～みたい (mitai)

Like; seems like (casual).

猫みたい。Like a cat.
Like / Similar (formal) ～のように / ～のような (no you ni / no you na)

Like; as if.

子どものようだ。Like a child.

Actions & Changes

Try doing ～てみる (te miru)

Try doing and see.

日本の映画を見てみる。Try watching a Japanese movie.
Do in advance ～ておく (te oku)

Do something for later.

予約しておく。Make a reservation in advance.
Start to ～はじめる (hajimeru)

Begin to ….

雨が降り始めた。It started to rain.
Difficult to ～にくい (nikui)

Hard to ….

食べにくい。Hard to eat.
Without doing ～ないで (naide)

Without doing ….

朝ご飯を食べないで学校へ行く。Go to school without eating breakfast.
Decide to ～ことにする (koto ni suru)

Decide to ….

毎日走ることにする。Decide to run every day.
It has been decided ～ことになる (koto ni naru)

It’s been decided that ….

来月、東京へ転勤することになった。It’s been decided that I’ll transfer to Tokyo next month.

Conditionals & Extent

The more… the more… ～ば～ほど (ba ~ hodo)

The more X, the more Y.

練習すればするほど上手になる。The more you practice, the better you get.
Should do ～ばいい (ba ii)

You should just do ….

分からない時、人に聞けばいい。If you don’t understand, you should just ask someone.
Should have done ～ばよかった (ba yokatta)

It would have been better if ….

もっと早く行けばよかった。I should have gone earlier.
To the extent that ～ほど (hodo)

To the extent that; so much that.

驚くほど速い。Surprisingly fast.
Not as … as ～ほど～ない (hodo ~ nai)

Not as X as Y.

去年ほど寒くない。Not as cold as last year.
Even if / No matter how ～ても (temo)

Even if …; no matter ….

雨が降っても行く。Even if it rains, I’ll go.
No matter how much ～どんなに～ても (donna ni ~ temo)

No matter how ….

どんなに忙しくても電話する。No matter how busy I am, I’ll call.
Not only… but also ～ばかりでなく (bakari de naku)

Not only … but also ….

肉ばかりでなく、野菜も食べる。Not only meat, but also vegetables.
On one hand… on the other ～一方で (ippou de)

On the other hand.

一方で、賛成の人もいる。On the other hand, there are people who agree.
More and more ～一方だ (ippou da)

Continues to ….

問題は複雑になる一方だ。Problems are getting more and more complex.

Advice, Tendency & Attitude

Should do ～べき (beki)

Should do; ought to.

もっと勉強すべきだ。You should study more.
Should not ～べきではない (beki dewa nai)

Should not do.

嘘をつくべきではない。You should not lie.
Should (advice) ～ことだ (koto da)

You should do ….

健康のために寝ることだ。You should sleep for your health.
Tend to ～がち (gachi)

Tends to be/do (often negative).

遅れがちだ。Tends to be late.
Hard to ～がたい (gatai)

Hard to (emotionally).

信じがたい。Hard to believe.
Even though / Despite ～くせに (kuse ni)

Even though; despite (blaming).

知っているくせに教えない。Even though you know, you won’t teach me.
Just as is ～まま (mama)

Leave in a state; as it is.

眼鏡をかけたまま寝る。Sleep with glasses on.
As if ～まるで～ようだ (maru de ~ you da)

Just as if; like.

まるで夢のようだ。It’s just like a dream.
Instead of ～代わりに (kawari ni)

Instead of.

私の代わりに行く。Go in my place.
While / Although ～ながらも (nagara mo)

While/although ….

貧乏ながらも幸せだ。While poor, (I’m) happy.
Pretend ～ふりをする (furi o suru)

Pretend to be/do.

知らないふりをする。Pretend not to know.
Suitable for ～向き (muki)

Suitable for.

初心者向き。Suitable for beginners.
Intended for ～向け (muke)

Intended for; aimed at.

子供向けの番組。A program aimed at children.
Full of / All over ～だらけ (darake)

Full of (often negative).

間違いだらけ。Full of mistakes.

Completion & Emphasis

Finish completely ～切る (kiru)

Do something completely.

読み切る。Finish reading completely.
Can’t finish ～切れない (kirenai)

Can’t finish; too much.

食べきれない。Can’t finish eating.
Only just ～きり (kiri)

Only; just (and nothing since).

一度きり会わない。Meet only once.
Hardly ever ～めったに～ない (metta ni ~ nai)

Hardly ever; rarely.

めったに行かない。Hardly ever go.
Never / By no means ～決して～ない (kesshite ~ nai)

Never; by no means.

決して忘れない。I will never forget.
Not really ～べつに～ない (betsu ni ~ nai)

Not particularly; not really.

別に好きじゃない。Not particularly fond.

Reason, Cause & Emphasis

No wonder / That explains ～わけだ (wake da)

That explains why; logically it follows.

彼は今パリにいるわけだ。So that explains why he is in Paris now.
Unreasonable that / wonder ～わけがない (wake ga nai)

No way; cannot be.

彼が嘘をつくわけがない。There is no way he would lie.
Not the case that ～わけではない (wake dewa nai)

Not necessarily; not entirely the case.

反対というわけではない。I’m not necessarily opposed.
Because (since) ～以上 (ijou)

Since; now that.

約束した以上、守るべきだ。Since you promised, you should keep it.
Precisely because ～だけあって (dake atte)

As expected of; because it is ….

彼女はプロだけあって上手だ。She is good, as expected of a pro.
Due to / Because (excuse) ～ものだから (mono dakara)

Because; giving a reason/excuse.

遅れたものだから、ごめんなさい。Sorry I’m late—because (something happened).
Only because ～ばかりに (bakari ni)

Just because of … (often negative result).

お金がないばかりに、行けなかった。Just because I had no money, I couldn’t go.

Time, Occasion & Manner

At the time / On the occasion ～際 (sai)

On the occasion of; when (formal).

ご利用の際は…When using (this service)…
As / In the capacity of ～として (toshite)

As; in the role of.

学生として参加する。Participate as a student.
Every time ～たびに (tabi ni)

Every time X happens.

会うたびに元気になる。Every time we meet, I cheer up.
For the first time in … ～ぶりに (buri ni)

For the first time in (a period).

10年ぶりに会う。Meet for the first time in 10 years.
Continue to ～ばかりだ (bakari da)

Keeps doing; continues to.

物価は上がるばかりだ。Prices just keep rising.
Approximately (time/amount) ～ばかり (bakari)

About; approximately (N2 usage).

1時間ばかり待った。Waited about an hour.
Immediately after ～か～ないかのうちに (ka ~ nai ka no uchi ni)

As soon as; barely before/after.

着くか着かないかのうちに雨が降り出した。It started raining as soon as (I) arrived.
Or / Possibly ～あるいは (aruiwa)

Or; possibly.

バスあるいは電車で行く。Go by bus or train.
Act as / Pretend to be ～ふりをする (furi o suru)

Pretend; act like.

知っているふりをする。Pretend to know.

Contrast & Unexpected

Far from … ～どころか (dokoro ka)

Far from; let alone.

独りどころか、家族もいる。Far from being alone, I have a family too.
Not at all ～ちっとも～ない (chitto mo ~ nai)

Not at all; not in the least.

ちっとも疲れていない。Not tired at all.

Emphasis, Assertion & Formality

Cannot help but… ～ずにはいられない (zu wa irarenai)

Cannot help but do; can’t refrain from.

泣かずにはいられなかった。I couldn’t help but cry.
In order to / For the purpose of ～べく (beku)

In order to (formal).

成功すべく努力する。Strive in order to succeed.
Must not ～べからず (bekarazu)

Must not; should not.

入るべからず。Must not enter.
Only because ～ばこそ (ba koso)

Precisely because.

あなたを愛しているばこそ言うのだ。I say this precisely because I love you.
If… then it’s over ～ばそれまでだ / ～たらそれまでだ

If X happens, that’s the end of it.

金がなくなればそれまでだ。If you run out of money, that’s it.
Only then / As long as ～なくしては (naku shite wa)

Without; if not for.

努力なくしては成功しない。Without effort, you won’t succeed.
To the end; absolutely ～あくまでも (akumade mo)

To the end; persistently.

あくまでも自分の意見を通す。Stick to your opinion to the end.
Dare to; deliberately あえて (aete)

Dare to; deliberately.

あえて危険をおかす。Deliberately take a risk.
In advance; beforehand あらかじめ (arakajime)

In advance.

あらかじめ準備する。Prepare in advance.
Just as one thought あんの定 (an no jou)

Just as expected; sure enough.

あんの定、雨だった。Just as I thought, it rained.
Which can exist only due to ～あっての (atte no)

Exists only thanks to.

あなたがあっての私だ。I exist only because of you.
Even; not even ～だに (dani)

Even; not even.

夢にだに思わなかった。Didn’t even dream of it.

Concessive & Extreme

Let alone / Not to mention ～はおろか (wa oroka)

Let alone; not to mention.

漢字はおろか、ひらがなも読めない。Can’t read hiragana, let alone kanji.
Must be; no doubt ～に違いない (ni chigainai)

Must be; surely.

彼は幸せに違いない。He must be happy.
Unbearable / Unworthy of ～にたえない (ni taenai)

Unbearable; not worth watching/reading.

見るにたえない。Unbearable to watch.
Unless / Only after ～てからでないと (te kara denai to)

Cannot do Y until X is done.

よく勉強してからでないと合格できない。Can’t pass unless you study hard.
While; at the same time ～がてら (gatera)

While doing X, also do Y.

散歩がてら、買い物をする。Go shopping while taking a walk.
Act as / Pretend to be ～ぶる (buru)

Act like; pretend to be.

偉ぶる。Act important.
Seeming / -like ～びる (biru)

Seem to be; appear (suffix).

大人びる。Act grown-up.
Manner / Way ～ぶり (buri)

Manner; way of doing.

話しぶり。Way of speaking.
Bound to happen ～べくして (beku shite)

As is bound to; naturally.

失敗すべくして失敗した。He failed as was bound to happen.
